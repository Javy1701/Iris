All other bracket tags in the database are to be ignored for AI prompt generation. These 10 terms were selected because they align best with how AI interprets sensory or appearance-based color language.

- Fallback Logic
If a color has only one of the approved bracket terms, include just that one.
If a color has none of the approved bracket terms, you will still generate a color prompt using HEX and LCh values and assign either 'Warm' or 'Cool' based on hue family.
